Constraints

● The product must be cross compatible (Windows, OSX, Linux), to allow all students to have easy access to the simulator.

● Simulator should be able to run through batch commands or user interface.

● User must install Cygwin, X11 and Unity application on Windows OS for compatible of the game.

● User must install Xcode, Xquartz and Unity application on Mac OS for compatible of the game
